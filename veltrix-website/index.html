Extension setup

Install in 4 steps

No account. No API key to configure. Load unpacked in Chrome and you're protected immediately.

01
Download & extract the ZIP

Download the extension ZIP above and unzip it to a local folder. Keep all files — do not rename anything inside.

veltrix-extension/
02
Open Chrome extensions page

Navigate to the extensions manager and enable the Developer Mode toggle in the top-right corner.

chrome://extensions
03
Load Unpacked

Click "Load Unpacked" and select the extracted folder. The Veltrix shield icon appears in your toolbar.

Load Unpacked → select folder
04
Set your backend URL (optional)

By default the extension points to localhost. For a hosted backend, update one constant.

background.js → API_BASE_URL
background.js
// Local dev (default)
const API_BASE_URL = "http://localhost:8000";

// Hosted backend on Render:
// const API_BASE_URL = "https://veltrix-api.onrender.com";
file structure
veltrix-extension/
├── manifest.json   ← Manifest v3 · permissions
├── background.js   ← Service worker · URL blocking
├── content.js      ← Gmail MutationObserver
├── popup.html      ← Toolbar popup UI
├── popup.js        ← Manual scan logic
└── blocked.html    ← Blocked URL warning page
Android setup

Install Veltrix App

Sideload the APK in three steps. No Play Store required.

01
Allow unknown sources

Go to Settings → Security → enable "Install unknown apps" for your browser or file manager.

Settings → Security → Unknown Apps
02
Download & install the APK

Download the APK above, tap the file in your notifications, and confirm install. No account needed.

apps/veltrix-app.apk
03
Share any text to scan

In SMS, WhatsApp, or Gmail — tap Share → Veltrix App. Text is analysed instantly and a risk score returned.

Share → Veltrix App
