Install

Extension setup

Load the extension unpacked in Chrome. No account or API key needed to try locally.

01
Download & extract the ZIP

Download the extension ZIP above and unzip it to any local folder. Do not rename files inside.

veltrix-extension/
02
Open Chrome extensions & enable Dev mode

Navigate to the extensions page, enable the Developer Mode toggle in the top-right corner.

chrome://extensions
03
Load unpacked

Click "Load Unpacked" and select the extracted veltrix-extension/ folder. The shield icon appears in your toolbar.

Load Unpacked → select folder
04
Set your API endpoint

By default the extension points to localhost. For a hosted backend, update one line:

background.js → API_BASE_URL
veltrix-extension / background.js 
// Local development (default)
const API_BASE_URL = "http://localhost:8000";

// Production — replace with your Render URL
// const API_BASE_URL = "https://veltrix-api.onrender.com";
file structure 
veltrix-extension/
├── manifest.json    ← Manifest v3 permissions
├── background.js    ← Service worker · URL blocking
├── content.js       ← Gmail MutationObserver
├── popup.html       ← Toolbar popup UI
├── popup.js         ← Manual scan logic
└── blocked.html     ← Blocked URL warning page
Android app

Mobile setup

Sideload the APK. Enable unknown sources once, install, done.

01
Allow unknown sources

Go to Settings → Security → enable "Install unknown apps" for your browser or Files app.

Settings → Security → Unknown Apps
02
Download & install APK

Download the APK above, tap the file notification, confirm install. No account or sign-in required.

veltrix-mobile.apk
03
Share any text to scan

In SMS, WhatsApp or Gmail — tap Share → Veltrix AI. The text is analysed and a risk score returned in seconds.

Share → Veltrix AI
